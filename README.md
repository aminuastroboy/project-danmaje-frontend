# Project Dan Maje Frontend

Mobile-first Next.js frontend for **Project Dan Maje**, a digital vending platform by **Astrovia Systems**.

## Stack
- Next.js
- React
- TypeScript
- CSS
- lucide-react

## Pages included
- Home dashboard
- Buy Data
- Buy Airtime
- Electricity
- Cable TV
- WAEC / NECO
- JAMB ePIN
- Recharge to Cash
- Wallet
- Fund Wallet
- Withdraw
- Transactions
- Agent dashboard
- Profile
- Notifications

## Local setup
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Push this project to GitHub.
2. Import the repo into Vercel.
3. Framework preset: **Next.js**
4. Build command: `next build`
5. Output handled automatically by Vercel.
